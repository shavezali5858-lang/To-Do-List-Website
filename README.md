# 📝 To-Do List Website

## 📌 Overview

This project is a simple and interactive to-do list website that helps users manage their daily tasks efficiently. Users can add, delete, and mark tasks as completed. The website updates dynamically without reloading the page.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Saves tasks using local storage (

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

1. User enters a task in the input field
2. Clicks the add button
3. Task is displayed in the list
4. User can mark it as complete or delete it
5. (If implemented) Tasks are saved in local storage

---



---

## 🧑‍💻 How to Run Locally

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
